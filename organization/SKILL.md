# 조직관리

## 구성원 (Member)

### 엔티티

- 이름, 이메일, 사번 (고유)
- 소속 부서 (선택), 직책 (선택)
- 입사일
- 역할: admin | manager | user
- 상태: 재직 | 휴직 | 퇴직
- 앱 초대 여부

### 비즈니스 규칙

- 구성원은 인증 시스템의 user와 연결된다 (초대 전이면 미연결)
- 사번은 시스템이 자동 채번한다
- 하나의 부서에만 소속될 수 있다
- 하나의 직책만 가질 수 있다
- 초대 발송 시 이메일로 초대 링크가 전달된다
- 초대 수락 시 계정이 자동 생성되고 세션이 시작된다

## 부서 (Department)

### 엔티티

- 이름
- 리더 (구성원 중 1명, 선택)
- 정렬 순서

### 비즈니스 규칙

- 플랫 리스트 구조 (하위부서 없음)
- 부서별로 리더를 지정할 수 있다
- 구성원을 부서에 배정/해제할 수 있다
- 드래그 앤 드롭으로 정렬 순서를 변경할 수 있다

## 직책 (Position)

### 엔티티

- 이름 (고유)
- 정렬 순서

### 비즈니스 규칙

- 구성원에게 직책을 배정/해제할 수 있다
- 드래그 앤 드롭으로 정렬 순서를 변경할 수 있다
